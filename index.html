P U B L I C -C L O U D
Pool aus virtuellen Ressourcen
 Funktioniert mithilfe von Internet
 Entwickelt aus Hardware
 Wird von Drittunternehmen verwalten
 Wird über Self-Service-Schnittstelle für mehrere Kunden automatisch bereit gestellt
 Damit eine Cloud müssen die Technologien nicht nur miteinander funktionieren sondern Integriert werden

Vorteile einer Public Cloud

  • Geringere anfängliche Einrichtungs- und Instandhaltungskosten als bei der Private Cloud
  • Ständig verfügbare und unendliche Skalierbarkeit
  • Die Sicherheit ist Sache von Drittanbietern
